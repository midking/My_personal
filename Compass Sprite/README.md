# Compass Sprite Image 만들기

### sprite image는 어떻게 만들어져 왔지?  
- 포토샵  
- N-MET  

위에서 하는 작업들은 단순하게 여러개의 image를 하나의 image로 합쳐주는 일을 합니다.  

### 단점은?  
- 시간이 오래걸림
- 유지보수가 힘듬
- css를 따로 작성

### compass sprite image를 사용하면?  
- sprite image를 자동으로 생성
- css 코드를 자동으로 생성
- 유지보수가 쉬움

만들어진 후 사용하고자 하는 장소에 class="이미지명" 만 입력하면 됩니다.
